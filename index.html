Handlebars Tutorial

Let's learn how we can create large blocks of HTML and insert them into HTML documents. We'll be using Handlebars, which has excellent docs and a tutorial to help you. Learn more here here.

Insert the script https://cdnjs.cloudflare.com/ajax/libs/handlebars.js/4.0.12/handlebars.min.js in the <head> of this document, before the main.js script. Don't forget to add the defer attribute.
